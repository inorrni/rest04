## 2026-06-09 — 초기 구현

### 기술 결정
- **스택**: Vite 5 + React 18 + 순수 CSS (Tailwind 미사용, CSS 변수로 토큰화)
- **라우팅**: 없음. 단일 페이지 스크롤 앵커 방식 (`href="#section-id"`)
- **다크모드**: `data-theme` 속성 + CSS 변수 오버라이드. 깜빡임 방지를 위해 `<head>` 인라인 스크립트로 초기화.
- **폰트**: Pretendard Variable (CDN, 한국어 최적화)

### 컬러 팔레트 (5색)
| 이름 | 값 | 용도 |
|------|-----|------|
| Sky Blue | `#0EA5E9` | 주 액션, CTA, 링크 |
| Eco Green | `#16A34A` | 보조 액션, 환경 지표 |
| Deep Navy | `#0C1A2E` | 다크 배경, 주요 헤딩 |
| Slate Gray | `#64748B` | 본문 텍스트, 보더 |
| Sage Mist | `#F0FDF4` | 섹션/카드 배경 |

다크모드: `--primary: #38BDF8`, `--secondary: #4ADE80` (더 밝은 변형)

### 파일 구조
```
src/
  data/
    site.js       ← 회사 정보·서비스·통계·문의유형
    videos.js     ← 유튜브 영상 데이터 (8 AI + 8 리터러시)
  lib/
    inquiries.js  ← Contact 폼 전송 (stub + Supabase 교체 코드)
  styles/
    index.css     ← :root 토큰 + 다크모드 + 전체 컴포넌트 스타일
  components/
    Nav.jsx       ← 고정 상단 바, 다크/라이트 토글, 모바일 드로어
    Hero.jsx      ← 풀스크린 히어로
    Services.jsx  ← 2×2 카드 그리드
    About.jsx     ← 좌우 분할 레이아웃 + 통계 카드
    Videos.jsx    ← 카테고리 탭 + 3×2 그리드 + 페이지네이션
    Contact.jsx   ← 클라이언트 검증 폼
    Footer.jsx    ← 3단 푸터
```

### 알아둘 함정
- **OG 이미지**: 반드시 JPG (RGB, 알파 없음) — `gen-assets.mjs`에서 `.flatten().jpeg()` 처리
- **GitHub Pages base**: `vite.config.js`에서 `command === 'build'` 시 `/rest04/` 적용
- **YouTube placeholder**: `youtubeId`가 `placeholder_`로 시작하면 "영상 준비 중" UI 표시
- **다크모드 flash**: `index.html` `<head>`의 인라인 스크립트가 `localStorage` 테마를 즉시 적용

### Contact 폼 검증 필드
- 이름: 필수
- 이메일: 필수 + 형식 검증
- 회사명: 선택
- 문의유형: 필수 (select)
- 내용: 필수, 10자 이상
- 상태: `idle → sending → success | error`
